# 在线视频下载工具

> 适用于网络在线点播视频下载（hls），仅用于学习交流

# 使用说明

首先在设置页面添加本地存储路径，然后在下载页面中搜索想要下载的资源，待视频加载完成之后，在首页右上角会出现可以下载的视频资源，选择时候点击立即下载即可
